# Resume tailoring instructions

Edit this file to control how the resume is tailored per job. It is injected
into the LLM prompt verbatim.

- Reorder and reword bullet points to surface the experience most relevant to the
  job description, and weave in the posting's key terminology where truthful.
- Prioritize quantified achievements (metrics, scale, impact) that match the role.
- Keep the same overall length as the base resume (one page if the base is one page).
- Match the seniority/tone of the target role.

Hard rules (never violate):
- Do NOT invent experience, employers, job titles, dates, degrees, or skills.
- Only rephrase/reorder what already exists in the base resume.
- Preserve the base document's \documentclass and \usepackage lines exactly.
